# paletter
Показывает доминирующий цвет изображения и палитру.

```js
Paletter.getColors(url, callback, colorsQuntity)
```

**url** — ссылка на изображение
**colorsQuantity** — количество цветов в палитре. Дефолт и максимум — 6 цветов.
